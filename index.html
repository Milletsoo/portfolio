Vibe Coding 作品

以下作品均由我独立完成产品设计，并使用 AI 编程工具（Claude Code）亲手开发实现。点击卡片下方链接可体验产品MVP。

扒舞神器

上传舞蹈视频 → AI 节拍检测 → 系统化拆解练习

📋 背景

舞蹈学习者看到喜欢的成品舞想自学，但肉眼很难一拍一拍数清楚动作，来回拖进度条又累又低效。

💡 解决的痛点

自动提取音轨分析 BPM 和拍点时间轴，按 4/4 拍结构每 8 拍自动分组，生成可视化"节拍网格"。每个拍点截一帧画面按 1-8 编号排版成幻灯片式分解图谱，搭配可调速（0.5–1.2x）+ 分段 AB 循环的播放器，把"反复拖进度条"变成"系统化拆解练习"。

Claude Code 提醒插件

再也不必守在 CC 窗口前等待 —— 权限请求 / 任务完成，实时弹窗通知

📋 背景

用 Claude Code 进行 Vibe Coding 时，最大的痛点是不知道 CC 什么时候卡在了权限请求、什么时候任务完成。代码小白尤其看不懂 CC 在问什么权限，只能守在窗口前干等。

💡 解决的痛点

制作了一个插件，在 CC 每次发出权限请求时自动弹窗通知，并把英文的技术性权限描述翻译成"大白话"，让代码小白也能一眼看懂 CC 在问什么、要不要同意。任务完成时也会推送通知，彻底解放等待时间。

清理电脑 Skill

小白也能安全清理电脑 —— AI 深度扫描 · 高风险预警 · 一键释放空间

📋 背景

电脑小白不懂电脑环境，下载软件后经常被连带安装垃圾软件，安装好后也不知道需要删除安装包来节省空间。时间一长，电脑越来越卡，却不敢随便删东西——怕误删重要文件。

💡 解决的痛点

用 Claude Code 的 Skill 功能，一句话就能让 AI 深度扫描电脑中的临时文件、缓存、重复文件和残留安装包。智能分类展示清理建议，并对高风险操作发出预警，防止误删重要文件。像有个懂电脑的朋友在旁边帮你把关。
